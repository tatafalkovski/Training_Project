Properties flex-elements

Свойства flex-элементов

Свойство Align-self -- переопеделяет выравнивание. Рабоатет как align-items, только применяется непосредственно к flex-элементам

Flex-basis базовый размер элемента

flex-basis: auto; -- значение по умолчанию

Помимо значения по умочлачнию можно указывать размер в %, px, и тд

Flex-grow -- контролирует возможность увеличиваться относительно бызового размера (flex-basis)

Flex-grow: 0; -- Значение по умолчанию

Если указать значение 1 тогда элементы заполнят флекс-контейнер по ширине

Flex-shrink -- контролирует возможность уменьшатся в размере. Значение по умолчанию 1, т.е. ему разрешено уменьшатся в размере. Если устонвить значение 0, тогда флекс-элементы не смогут уменьшаться в размере меньше чем указано базовое(flex-basis) значение.

Более компактная запись выглядит таким образом:

flex: 0 1 auto; -- 0-grow, 1-shrink, auto-basis
